# 求職者スカウト登録・検索サイト

## 概要
人材が自分の情報を登録、企業が検索しスカウトメッセージを送れます。

## コンセプト
今後の拡張に向けた基本機能の構築。

## バージョン
Ruby 2.6.2 Rails 5.2.3

## 機能一覧
 ログイン機能
 ユーザー登録機能
 人材一覧検索機能
 人材検索機能
 メッセージ受送信機能

 企業アカウント登録機能
 企業アカウントログイン機能
